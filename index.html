json_loader overview

Json loader is a program that loads files in from TileD so you can incorporate them into a 2D tile game.

View details »

Using json_loader

First, create a map using TileD and save it. Next create a program to load the file in a seperate directory. Once you have loaded the map file you can manipulate the program to do what you want.
